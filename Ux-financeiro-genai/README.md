# Experiência Digital de Relacionamento Financeiro com GenAI 

Este projeto consolida os conhecimentos em **IA, Python e UX Design**, criando uma interface de assistente financeiro personalizada.

## Princípios de UX Aplicados
* **Clareza:** Linguagem simples e direta para explicar produtos complexos (como CDB).
* **Transparência:** Cálculos demonstrativos com notas explicativas sobre as taxas.
* **Personalização:** O assistente utiliza o nome do usuário e mantém o contexto da simulação.

## Funcionalidades
1. **FAQ Inteligente:** Respostas rápidas sobre termos bancários usando mapeamento de intenção.
2. **Simulador de Investimentos:** Algoritmo em Python para projeção de ganhos.
3. **Persistência de Contexto:** Fluxo de conversa que guia o usuário do interesse à simulação.

## Tecnologias
* **Python 3:** Lógica de negócio e cálculos.
* **GenAI (Conceitual):** Estrutura preparada para integração com APIs de LLM (ChatGPT).
